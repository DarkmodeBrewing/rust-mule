# API Helper Scripts

Endpoint-focused helper scripts used by documentation and manual API verification.

Examples:
- `health.sh`, `status.sh`, `events.sh`
- `kad_search_keyword.sh`, `kad_publish_keyword.sh`
- `download_create_from_hash.sh` (hash-first download initiation helper)
- `debug_routing_summary.sh`, `debug_probe_peer.sh`

These are not full scenario test harnesses; they are single-purpose API helpers.
